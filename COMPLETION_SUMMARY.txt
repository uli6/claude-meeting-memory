  MIGRATION OVERVIEW

FROM:                                   TO:
  Google OAuth                          Himalaya IMAP CLI
  Google Calendar API                   Plann CalDAV CLI

  ✅ Open-source tools (user-controlled)
  ✅ Better context (emails + memory)
  ✅ Decentralized (self-hosted compatible)

================================================================================
  SYSTEM ARCHITECTURE
================================================================================

Two Operating Modes:

MODE 1: AUTOMATIC CALENDAR MONITORING (OPTIONAL)
  Every 10 minutes:
    Plann Calendar → calendar_watcher.sh → Cache → Slack (optional)

MODE 2: ON-DEMAND MEETING BRIEFING (ALWAYS AVAILABLE)
  User: /pre-meeting "Sales Review"
    → meeting_prepper.py
    → Himalaya (emails) + Memory + Plann (cache)
    → Claude API (briefing)
    → User gets: Pending Items + Context + Email Context

Information Flow:
  Himalaya (IMAP)     Plann (CalDAV)     Memory System
    ↓                    ↓                    ↓
    └────────────────────┬────────────────────┘
                         ↓
                  Pre-Meeting Skill
                         ↓
                Briefing Generation
                         ↓
    🔥 Pending Items | 📚 Context | 📧 Emails
